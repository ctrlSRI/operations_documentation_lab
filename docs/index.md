# Documantation Index

This repository documents practical system administration workflows, explanations and reference material for a local operations lab enviroment.

The current focus is on Windows Server administration with Hyper-V, Powershell, Active Directory, DNS, DHCP, Group Policy, and file services.
The lab will later be extended with Ubuntu Server documentation to cover Linux-based operations workflows as well.

## Windows Server documentation

The Windows Server section covers the setup and administration of a Windows Server lab environment.

Planned topics include:

- Hyper-V
- Windows Server installation
- PowerShell
- Active Directory
- DNS
- DHCP
- Group Policy
- File services
- Troubleshooting

## Goals

- Hyper-V setup
- Virtual networking
- Windows Server installation
- PowerShell-based system inspection
- Basic network troubleshooting
- Active Directory and DNS setup planned

## Ubuntu Server documentation

The Ubuntu Server section will cover Linux-based operations workflows.

Planned topics include:

- Ubuntu Server installation
- Linux command line basics
- Users and groups
- File permissions
- SSH
- Services with systemd
- Logs with journalctl
- Basic troubleshooting
